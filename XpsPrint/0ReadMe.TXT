This sample demonstrates how to use Aspose.Words for .NET to print documents using 
the XpsPrint API (http://msdn.microsoft.com/en-us/library/dd374565(VS.85).aspx).

This sample will run on either Windows 7, Windows Server 2008 R2 or Windows Vista.

To run on Windows Vista you need to have the Platform Update for Windows Vista installed
(http://www.microsoft.com/downloads/details.aspx?FamilyId=081c218f-c845-47d1-b124-71f80bf21638&displaylang=en).
